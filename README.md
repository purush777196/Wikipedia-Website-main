**Wikipedia Search Application**

  A simple and responsive Wikipedia Search app that allows users to search for articles by keyword and view results instantly using the Wikipedia Search API.

**Features**

  Search Wikipedia articles by keyword
  
  Fetches data using Fetch API
  
  Displays a loading spinner during API calls
  
  Shows article title, link, and description
  
  Opens results in a new tab
  
  Responsive and user-friendly UI

CSS

JavaScript

Fetch API
